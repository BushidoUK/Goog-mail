# Goog-mail.py OSINT tool
## Usage
After cloning the repo (or downloading the script), run the script by typing

```
./goog-mail.py <domain>
```

For example:

```
user@box:~# python goog-mail.py example.com
```

or

```
user@box:~# ./goog-mail.py example.com
```
